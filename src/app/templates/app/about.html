What is SPDX?

System Package Data Exchange (SPDX) is a standard format for communicating the components, licenses, copyrights, provenance, and security information associated with systems and software packages.

What is SPDX Online Tools?

A portal for SPDX SBOM and license management. Validate, convert, compare, and edit SPDX documents. Identify an SPDX License ID from license text and a submit new license to the SPDX License List.
